Keep Ubuntu or Debian up to date

To keep your stable version of Debian up to date, open a terminal and execute these commands in the following order:

sudo apt-get update
sudo apt-get upgrade
sudo apt-get dist-upgrade
sudo apt-get autoremove

Finally, reboot your system if Debian tells you to do it or if you want maximum guarantee. Restarting the updated applications or logging out and then logging in is enough.
