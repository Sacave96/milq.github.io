Install OpenCV on Ubuntu or Debian

Note: This tutorial has been tested on Ubuntu 12.04 LTS with OpenCV 2.4.3.

Install OpenCV on Ubuntu or Debian is a bit long but very easy. Simply, follow these steps:

1. UPDATE AND UPGRADE UBUNTU

Open your terminal and execute in the following order:

sudo apt-get update
sudo apt-get upgrade
sudo apt-get dist-upgrade
sudo apt-get autoremove

2. INSTALL THE DEPENDENCIES

Now install the dependencies:

Build tools:

sudo apt-get install build-essential cmake

Image libraries:

sudo apt-get install libjasper-dev libjpeg-dev libopenexr-dev libtiff4-dev

Audio libraries:

sudo apt-get install libfaac-dev libvorbis-dev

Video libraries:

sudo apt-get install libtheora-dev libx264-dev libxvidcore-dev libv4l-dev

GUI:

sudo apt-get install qt5-default libgtkglext1-dev

Java:

sudo apt-get install ant default-jdk

Parallelism libraries:

sudo apt-get install libtbb-dev

Linear algebra libraries:

sudo apt-get install libeigen3-dev

Python:

sudo apt-get install python-dev python-numpy python-tk

3. DOWNLOAD AND DECOMPRESS OPENCV

Enter the OpenCV official website and download the latest version for Linux. Then decompress the downloaded file.

4. COMPILE OPENCV

In your terminal, make sure you are within the OpenCV directory and run the following commands:

mkdir build
cd build
cmake -DWITH_OPENGL=ON -DWITH_QT=ON -DBUILD_NEW_PYTHON_SUPPORT=ON -DWITH_TBB=ON -DWITH_V4L=ON -DBUILD_EXAMPLES=ON -DINSTALL_C_EXAMPLES=ON -DINSTALL_PYTHON_EXAMPLES=ON   ..
make
sudo make install

Now, you have installed OpenCV with the following extensions: 3D visualization, Python, Java, TBB, OpenGL, video, and Qt interface.

5. CONFIGURE OPENCV

In your terminal, execute:

sudo ldconfig

Finally, reboot your system. If you don't want to reboot, restarting the terminal or logging out and then logging in is enough.
