Things to do just after installing Debian

Text.

1. ENABLE SUDO ON YOUR USER ACCOUNT AND UPDATE DEBIAN

There are many benefits of using sudo instead of su root so it's important enable 'sudo' on your user account.

2. INSTALL ADDITIONAL SOFTWARE

Admin:

sudo apt-get install gdebi

Devel:

sudo apt-get install build-essential

GNOME:

sudo apt-get install nautilus nautilus-open-terminal file-roller gnome-font-viewer

TeX:

sudo apt-get install texlive-full texmaker

Net:

sudo apt-get install transmission-gtk

Go to Settings, Preferred applications and select Gnome Terminal and Nautilus as default applications.

3. INSTALL NON-FREE DRIVERS

Run sudo lswh | grep UNCLAIMED to see the unclaimed drivers. Verify all the drivers have their product and vendor tags. The commands lspci, lsusb can help you.

For example, for my Wi-Fi I download and install the non-free Realtek firmware package and for my ATI graphic card I download and install the non-free driver.

4. INSTALL NON-FREE SOFTWARE

    http://google.com/chrome
  • Chrome browser (the free alternative is Chromium.
  • Dropbox.
  • Sublime Text.
  • VirtualBox (some dependencies are non-free).

4. CUSTOMIZE THE LOOK OF YOUR DESKTOP

Install Murrine engine and their respective themes:

sudo apt-get install murrine-themes

This is my configuration in Debian:

  • Desktop Environment (DE): Xfce.
  • GTK Theme: Albatross.
  • Window Manager (WM): Albatross.
  • Fonts: Roboto and Liberation Mono (monospace). Download Normal, Bold and their italics. Alternative font: Cousine.
  • Icons: Faenza.

If you use Xfce, enable autologin, confirm your workspaces (Settings, Workspaces)...
